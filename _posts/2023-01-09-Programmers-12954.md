### Answer Code1

``` java
class Solution {
    public long[] solution(int x, int n) {
        long[] answer = new long[n];
        long ans = x;
        
        for(int i = 0; i< n; i++) {
            answer[i] = ans;
            ans += x;
        }
        for(int i = 0; i < n; i++) {
            System.out.print(answer[i]);   
        }
        return answer;
    }
}
```

### Answer Code2

``` java
import java.util.*;
class Solution {
    public static long[] solution(int x, int n) {
        long[] answer = new long[n];
        answer[0] = x;

        for (int i = 1; i < n; i++) {
            answer[i] = answer[i - 1] + x;
        }

        return answer;

    }
}
```
