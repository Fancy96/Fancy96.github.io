### Answer Code1(23.01.10)

``` java
class Solution {
    public long solution(int a, int b) {
        long answer = 0;
        if(a < b) {
            for(int i = a; i <= b; i++) {
                answer += i;
            }
        } else if(a > b){
            for(int i = b; i <= a; i++) {
                answer += i;
            }
        } else {
            answer = a;
        }
        return answer;
    }
}
```
