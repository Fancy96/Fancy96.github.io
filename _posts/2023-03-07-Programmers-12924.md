## Answer Code1(2023.03.07)

```java
class Solution {
    public int solution(int n) {
         int answer = 0;
          
          for(int i = 1; i <= n; i++) {
              int sum = 0;
              for(int j = i; j <= n; j++) {
                  sum += j;
                  
                  if(sum == n) {
                      answer++;
                      break;
                  } else if(sum > n) {
                      break;
                  }
              }
          }      
          return answer;
    }
}
```
